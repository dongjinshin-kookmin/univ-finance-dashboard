## 0. 데이터 기반과 인덱싱 계약 (Ground Truth)

모델은 아래 **실측 확인된** 스키마 위에서만 동작한다. (코드 직접 확인 결과)

### 0.1 columnar 인덱싱 계약
- `rows = [[schoolIdx, year], ...]` — 길이 **2088**(= 실제 자금계산서 존재 학교-연도). 이것이 마스터 조인 키.
- 다음 배열은 모두 `rows`에 **1:1 정렬된 병렬 배열**(길이 2088):

### 0.2 계정 계층 (수입측, 실측)

## 1. 핵심 난제 — 5110의 학부·대학원·정원외 분해 (모델의 심장)

### 1.1 문제 정의
입학정원(감축 레버)은 **학부 정원내**뿐인데, 5110/5112 수입에는 **대학원 수업료 + 정원외(외국인·편입)
수업료**가 섞여 있다. 이를 분리하지 않으면 정원 감축의 수입 영향이 크게 **과대추정**된다.

**실증(국민대 2024):**
- 파생단가 naive = 5110/재학생 = 156,606,369 / 14,486 = **10,811 천원** — 전국 학부 평균(중앙값 6,759)의 1.6배.
- 원인: 5112(154.7B)에 대학원 재학생 ~3,571명분 수업료가 포함되나, 분모(14,486)엔 학부만 있음.
- 대학원 환산 정리: `p_ug = 5112 / (H_ug + ρ_g·H_grad)` = 154,744,927 / (14,486 + 1.4×3,571)
  = 154,744,927 / 19,485 = **7,942 천원** → 현실적 학부단가로 교정됨. **차이 약 2,870천원(36%)이 대학원 오염분.**

이 오염 규모는 학교마다 다르다(대학원 큰 연구중심대 심함, 교육중심대 미미). 따라서 **학교별 보정 필수**.

### 1.2 세그먼트 정의 (수입·인원 동시 분해)
| 세그먼트 | 수입 | 인원(H, 재학) | 정원 감축 레버에 반응? |
|---|---|---|---|
| 학부 정원내 수업료 R_in | φ_in·5112 | H_ug_in | **예 (유일한 반응 세그먼트)** |
| 학부 정원외 수업료 R_out | φ_out·5112 | H_ug_out (외국인·편입) | 아니오(별도 정원) |
| 대학원 수업료 R_grad | φ_grad·5112 | H_grad (부설대학원) | 아니오 |
| 입학금 5111 | — | — | 사실상 0(폐지 완료), 외생 고정 |
| 수강료 5120 | — | — | 계절/시간제, 인원에 약결합(탄력성 낮게) |

φ_in + φ_out + φ_grad = 1 (5112 기준).

### 1.3 인원 매핑 (KESS → 세그먼트, 전부 추출 가능)
- H_ug (학부 재학, 정원내+정원외) = `재학생_전체_계`(base) — **이미 적재**(schools[].enroll).
- H_grad (대학원 재학) = Σ`부설대학원` 행 `재학생_전체_계` — **추출 필요**(신규).
- A_in (학부 정원내 입학자) = `정원내_입학자_전체_계` — **이미 적재**(ext.series.입학자수).
- A_out (학부 정원외 입학자) = `정원외_입학자_전체_계`(base) — **추출 필요**(신규).
- Q (학부 정원) = `입학정원_전체`(base) — **이미 적재**(ext.series.입학정원).
- H_ug_out (정원외 학부 재학 스톡) ≈ A_out × m_out (m_out ≈ 학부 배율, 편입/외국인 재원기간 유사; 기본 4.0).
- H_ug_in = H_ug − H_ug_out.

국민대 2024 예: A_out=496 → H_ug_out ≈ 496×4 ≈ 1,984 → H_ug_in ≈ 14,486−1,984 = **12,502**.
따라서 정원내 배율 m_in = 12,502/2,963 = **4.22** (naive 14,486/2,963=4.89보다 정확).

### 1.4 학부 단가 p_ug 추정 — 3방법(우선순위)
**방법 A(권장 폴백, 데이터 자급):** 대학원 환산 파생단가
```
p_ug = 5112 / (H_ug + ρ_g · H_grad)            [천원/인·년]
```
- ρ_g = 대학원/학부 등록금 배율. 전국 평균 근사 **1.4**(범위 1.2~1.7, 학교유형별 조정 가능). 확인 필요값.
- 유예생·정원외 외국인 할인은 2차 오차로 무시(주석).

**방법 B(우선, 확보 시):** 공시 학부 1인당 평균등록금 (data.go.kr 15107738 / 대학알리미).
- 공시단가는 학부·정원내 표준등록금(감면 전) → 5112의 학부분과 개념 정합. 확보되면 p_ug에 직접 사용.
- **교차검증(게이트):** 공시 p_ug로 역산한 대학원단가 `p_grad = (5112 − p_ug·H_ug)/H_grad`가
  ρ_g = p_grad/p_ug ∈ [1.2, 1.8]에 들면 통과. 벗어나면 인원·단가 소스 재점검.

**방법 C(그룹 폴백):** A·B 모두 결측인 소수 학교 → 동일 (설립유형·규모·계열) 그룹 중앙 p_ug 대입.

**φ_in 산출(반응 세그먼트 비중):**
```
R_in = p_ug · H_ug_in ,  R_grad = p_grad · H_grad ,  R_out = p_ug · H_ug_out
φ_in = R_in / 5112
```
(주의: 학부 정원내·정원외는 **같은 단가 p_ug**를 낸다. 오직 대학원만 다른 단가. 따라서 φ 분해의 불확실성은 사실상 ρ_g 하나에 집중된다.)

### 1.5 설계 결정 — "델타를 계산한다"
정원 감축의 수입 영향은 **오직 R_in의 변화**로만 발생한다. 나머지(R_out, R_grad, 5111, 5120)는
정책과 무관하게 기준 경로에 고정한다. 그래서 모델의 출력 핵심은 절대 예측이 아니라 **정책 델타**:
```
Δ등록금수입(t) = p_ug(t) · ΔH_ug_in(t)
```
이 프레이밍은 φ 분해·기준 경로의 불확실성을 상당 부분 상쇄한다(둘 다 델타에서 소거). **주력 지표로 델타를 표시**하고, 절대 예측은 보조로 둔다. (근거: 사용자 질문은 "감축하면 vs 안 하면"의 차이이며, 이는 델타로 정확히 답해진다.)

## 2. 코호트 전개 모델

### 2.1 잔존기간(재원) 모델 — 이론적 토대
정의: s_k = 한 정원내 신입 코호트가 입학 후 **k년째 캘린더연도에 재학(등록·납부) 상태로 기여하는 비율**
(k=0,1,2,...). 그러면 신입 A_in(c)에서 유래해 연도 t=c+k에 재학 중인 인원 = A_in(c)·s_k.

**Little의 법칙:** 정상상태(일정 신입 A)에서 재학 스톡 = A · Σ_k s_k. 즉
```
Σ_k s_k = m_in  (정원내 배율, 실측 가능)
```
→ **관측된 배율 m_in이 곧 평균 재원연수의 추정치**이며, 스케일과 전개 시간축을 동시에 고정한다.
중도탈락·유예·초과학기·군휴학의 **순효과가 m_in에 이미 반영**되어 있다(별도 파라미터 불요).
(실측: 학부배율 중앙값 4.035 ≒ 4년제 순수. 국민대 m_in≈4.22.)

### 2.2 잔존곡선 s_k — 3단계 설계

**(a) 균등코호트 근사 [학년별 데이터 없을 때, 기본]**
전면 사각형 + 기하꼬리:
```
s_k = 1                (k = 0,1,2,3)          ← 4년 정규 재학
s_k = g^(k-3)          (k ≥ 4)               ← 초과·휴학지연 꼬리
```
Σ s_k = 4 + g/(1−g) = m_in  ⟹  **g = (m_in − 4)/(m_in − 3)**.
- m_in=4.04 → g=0.038(꼬리 거의 없음, 딱 4년).
- m_in=4.89(국민대 naive) → g=0.47(느린 꼬리 = 높은 휴학 반영). 정원내 4.22 → g=0.18.
- 근거: 단일 관측량 m_in만으로 전개 속도까지 결정 → 과적합 없음, 학교별 자동 적응.

**(b) 휴학 보정 캘린더 스프레드**
휴학률 h(=휴학/재적, 중앙값 0.222)가 높을수록 정원 감축의 재학 반영이 **더 느리게, 더 길게** 퍼진다.
꼬리 decay g의 하한을 h와 연동해 보정: g ← max(g, h·κ) (κ≈1, 튜닝). 근거: 휴학이 많으면 한 코호트가
더 많은 캘린더연도에 흩어져 재학 → 전이가 완만.

**(c) 정밀 잔존곡선 — 패널 디컨볼루션 [정확도 상향, 데이터 허용 시]**
관측 H_ug_in(t) = Σ_k s_k·A_in(t−k)를 2016~2024 패널에 대해 제약회귀로 s_k 추정:
- 제약: s_k ≥ 0, s_0=s_1=s_2=s_3≈1(정규재학 근사), k≥4 단조감소, Σ s_k = m_in.
- 9개년은 학교별로는 불충분(과소결정) → **그룹 풀링**(설립유형×규모×휴학률 구간)으로 s_k 추정 후 학교 m_in로 스케일.
- 산출: 그룹별 s_k 벡터. 개별 학교는 (a) 파라메트릭을 기본, 그룹 s_k를 옵션으로.

**(d) 정밀 학년 코호트 [현재 불가 — 조건부 설계]**
학년별 재학·진급률·편입 유입이 확보되면:
```
E_g(t+1) = E_{g-1}(t)·ρ_{g-1→g} + T_g(t)     (진급률 ρ, 편입유입 T)
E_1(t) = A_in(t)                              (신입)
H_ug_in(t) = Σ_{g=1..G} E_g(t)
```
→ 대학알리미 학년별 API(인증) 확보를 **후속 과제**로 명시. 확보 전엔 (a)~(c)가 주력.

### 2.3 중도탈락·편입·군휴학 처리 (데이터 부재 대응)
- **중도탈락**: 학교별 값 미확보(인증 API 필요). 기본은 **m_in에 흡수**(관측 배율이 이미 순감소 반영).
  시나리오용 "이탈률 가중" 슬라이더는 s_k(k≥1)를 일괄 축소해 스트레스 부여(기본 0%).
- **편입 유입**: KESS에 컬럼 없음. 편입은 정원외로 취급 → **정원 감축과 무관하게 고정**. 단 m_in을
  "정원내 재학/정원내 입학자"로 계산하므로(H_ug_out 분리) 편입분이 델타에 잘못 곱해지는 오류 방지.
- **군휴학·초과학기**: s_k 꼬리 g와 휴학보정(b)으로 흡수. 남학생 많은 공대·예체능(높은 휴학) 자동 반영.

### 2.4 전개 마스터 방정식
정원 감축 정책: 시작연도 t0, 연차 감축률 r(t)(누적). 신규 정원 Q'(t)=Q(t)·(1−r(t)).
충원 시나리오로 신규 정원내 입학자 A'_in(t) 산출(3장). 델타:
```
ΔA_in(c) = A'_in(c) − A_in,base(c),   c ≥ t0   (c < t0 이면 0)
ΔH_ug_in(t) = Σ_{c=t0}^{t} ΔA_in(c) · s_{t−c}
Δ등록금수입(t) = p_ug(t) · ΔH_ug_in(t)                         [천원]
```
수입 롤업:
```
5112'(t) = 5112_base(t) + Δ등록금수입(t)
5110'(t) = 5111_base(t) + 5112'(t)
5100'(t) = 5110'(t) + 5120'(t)         (5120' = 5120_base·수강료탄력 반영, 2장·4장)
```
`p_ug(t) = p_ug(t0)·Π(1+π_τ)`는 단가 상승 시나리오(4장). 기준 경로 heads는 t0 고정(정책 델타 프레이밍).

## 3. 충원율 내생성

정원을 줄이면 선발 경쟁이 오르고 미충원 좌석이 먼저 사라져 충원율이 **오르는 경향**이 있다.
이를 무시하면(비례 감소) 미충원 지방대의 수입 손실을 **크게 과대추정**한다. 두 옵션 제공:

**옵션 1 — 보수적(충원율 고정):**
```
A'_in(t) = f0 · Q'(t) = A_in,base(t) · (1 − r(t))     (f0 = 기준 충원율)
```
입학자가 정원에 비례 감소 → **손실 최대**. (완전충원 학교엔 현실적, 미충원 학교엔 과대.)

**옵션 2 — 현실적(수요 앵커·부분회복) [권장 기본]:**
현재 입학자 A_in,base는 **실수요의 하한**. 미충원(f0<1)이면 좌석 초과 → 감축은 **빈 좌석부터 제거**:
```
A'_in(t) = min( A_in,base(t) ,  Q'(t) · f_ceiling(t) )
f_ceiling(t) = min( f_max ,  f0 + β·r(t) )            (β = 회복계수, f_max≈1.0)
```
- β=1(완전회복): A'_in = min(A_in,base, Q'). 손실 = max(0, r·Q − 미충원좌석). **첫 (Q−A_in)만큼의 감축은 손실 0.**
- f0≈1(완전충원): A'_in ≈ Q'·1 = A_in·(1−r) → 옵션 1과 수렴(당연: 채울 여석 없음).
- 근거: 국민대(f0=0.999) → 감축 ≈ 손실 1:1. 미충원 지방대(f0=0.85) → 15% 감축까진 실손실 미미.
  이 **비대칭**이 정책 함의의 핵심이며, min() 수요앵커 형태가 이를 정확히 포착.

기본값 β=0.5(부분회복), 슬라이더로 0(고정=보수)~1(완전회복=낙관).

### 3.3 학령인구 압력과 f0(t) 기준선 — v1 결정

배경: 소재 시도의 학령인구가 줄면 **미충원 지방대는 정책과 무관하게 f0(충원율)이 스스로 하락**한다.
반면 **완전충원 대학(f0≈1)은 초과수요 쿠션**으로 당분간 f0 유지가 합리적이다. 이 비대칭을 f0(t)
기준선에 반영할지가 쟁점.

**v1 결정: f0(t) 기준선은 최근 실측 충원율(예: 2024)로 고정(default). 인구 연동은 옵션 시나리오로 분리.**

근거:
1. **실측 충원율이 이미 인구 압력을 내재.** 2016~2024 관측 충원율 하락에 학령인구 감소가 이미 녹아 있다.
   f0(t0)=최근 실측을 기준선으로 두면 "현재까지의 인구 효과"는 자동 반영된다.
2. **델타 프레이밍에서 상당 부분 상쇄.** 감축·무감축 두 경로에 동일 f0 기준선이 적용되어 공통 하락분은
   델타에서 소거. 남는 것은 min() 수요앵커의 비선형 상호작용뿐.
3. **인구→특정 대학 충원율 전가율은 강한 가정.** 대학은 전국모집(수도권 상위대)~지역모집(지방대)
   스펙트럼이라 시도 인구 감소의 개별 충원율 전가율이 학교마다 크게 다르다. 일률 적용은 정확도 리스크
   → v1 기본에서 배제.

**옵션 시나리오 "충원율 하락 드리프트"(사용자 토글, 기본 OFF):**
```
f0(t) = f0(t0) · (1 + λ · d_sido(t))
d_sido(t) = pop18_sido(t)/pop18_sido(t0) − 1        (음수, 소재 시도 18세 감소율; 3.4 실적/보정 기준)
λ = 충원율의 인구 전가율(기본 0.5, 범위 0~1)
```
- **λ는 f0<1 학교에서만 실효.** 완전충원(f0≈1)은 f_ceiling 상한(min, ≤f_max)에 걸려 무반응 → **비대칭 자동 구현.**
- 즉 배출지 미충원 지방대만 f0가 추가 하락, 초과수요 대학은 유지 → 지적된 비대칭을 정확히 표현.
- 이 옵션의 불확실성(λ·d_sido)은 비관 밴드로 흡수. **v1 기본 화면은 f0 고정**이므로 "정책과 별개인 자연
  감소"는 드리프트 토글·주석으로만 노출(한계, 9장).

### 3.4 인구 데이터 사용 지침 — 실적 기준·추계 편향 보정 (필수)

D4 수집으로 **주민등록 실적** `build/interim/population_actual.csv`
(시도 × 2016~2024 × {18세 · 대학학령18-21 · 학령인구6-21}, 등록지 기준, 459행·결측 0)이 확보됐다.
기존 추계 `ext.population`(2022~2052, 거주지 기준)과의 사용 관계를 아래로 **고정**한다.

**핵심 발견(build/d4_report.md 실측):** 대학학령(18-21세) **추계가 실적을 계통적으로 과대**
(전국 2022 +4.59% / 2023 +4.01% / 2024 +2.87%, 시도 MAPE 9.05%·최대 29.56%). 원인은 개념 차이 —
**추계=거주지 기준(센서스), 실적=등록지 기준(주민등록).** 18-21세는 진학·군입대로 거주지≠등록지라

**지침:**
1. **역사 구간(2016~2024)은 반드시 실적(population_actual) 사용.** 지역 백테스트(6장)·f0 드리프트

3. **권장 프록시:** 절대 대학학령 대신 **편향 없는 학령인구(6-21) 추계의 감소율**을 지역 수요 트렌드로

4. **기존 위기스코어(ext 요소 ⑤)도 동일 편향에 노출** — 시뮬레이션 인구 프록시는 실적/보정값을 쓰고,

### 5.2 지출 대응 시나리오 (수지 악화 증폭)
```
ΔOP_EX(t) = Σ_j ε_j · (ΔH_ug_in(t)/H_ug_in(t0)) · OP_EX_j(t0)
```
- 인건비 4100: 경직. ε_L ∈ [0, 0.5], **기본 0.15**(단기 경직, 장기 0.3 조정 지연).
- 관리운영비 4200·연구학생경비 4300: ε ∈ [0.3, 0.6].
- 단일 슬라이더 "지출 연동 강도" γ∈[0,1]로 복합 탄력을 스케일(0=완전경직→수지 최악, 1=완전연동→수지 중립).

### 5.3 증폭 서사 (핵심 인사이트)
인건비가 경직(ε_L=0)이면 수입 Δ 전액이 수지로 전가되고, **분모(5100) 축소로 인건비부담률이
비선형 급등**한다. 예: 5100 −10%, 4100 불변 → 인건비부담률 = 4100/(0.9·5100), 즉 +11.1%p 상대상승.
"운영수지(ε_L=0 최악) vs (ε_L=0.3)" 두 선을 함께 표시해 고정비 리스크를 시각화.

## 6. 백테스트 설계 (정확도 검증)

### 6.1 분할·절차
1. **적합(2016~2020):** m_in, s_k(파라메트릭 g / 그룹 디컨볼루션), p_ug(2020), ρ_g를 적합.
2. **예측(2021~2024):** **실제 입학자 A_in(2016~2024)**을 모델에 투입 → H_ug_in(t)·5112(t) 예측.
3. **비교:** 예측 vs 실제 `enroll`(학부재학)·실제 5112·5110.

### 6.2 3층 검증 (오차 원천 분리)
- **L1 재학 전개:** 예측 H_ug_in vs 실제 학부재학. 잔존곡선 정확도. **목표 MAPE < 3%.**
- **L2 수입(인원 given):** 예측 5112 = p_ug·heads vs 실제 5112. 단가 드리프트 격리. **목표 MAPE < 4%.**
- **L3 종단 5110/5100:** (입학금 5111은 실제값 고정) 수업료 종단. **목표 MAPE < 5%.**

### 6.3 오차 지표·합격 기준
- 1차 **MAPE**(수입가중 집계 + 학교별 분포), **MPE**(편향, |MPE|<1.5% — 계통편향 금지), RMSE, 학교별 p90 오차.
- **Naive 대비 skill**(필수): B0=전년유지, B1=선형추세. `skill = 1 − MAPE_model/MAPE_naive`.
  **합격: skill > 0**(반드시 naive 초과), 권장 skill ≥ 0.3. 모델이 naive를 못 이기면 복잡도 정당화 실패로 간주.
- 합격표: L1<3% & L2<4% & L3<5% & |MPE|<1.5% & skill>0 → **통과**. 미달 시: (i) m_in 그룹 재적합,
  (ii) s_k 꼬리 g 재추정, (iii) p_ug 소스 A→B 교체, (iv) ρ_g 재보정 순으로 진단.

### 6.4 구조 단절·교란 처리 (정확도 함정)
- **입학금 폐지(2018~2022):** 5111 급감은 정원과 무관. 백테스트는 **5112 기준**으로 하고 5111은 외생(실제값) 고정.
- **COVID(2020~2022):** 휴학 급증·반수·편입 변동으로 재학 스톡 교란 → L1 오차 상승 가능. 해당 구간
  잔차를 **별도 표기**하고 합격 판정에서 민감도로 취급(제외 아님, 표시).
- **폐교·급감학교:** 소수 극단 배율(배율 평균 5.99 vs 중앙 4.04의 우측꼬리)은 수입가중 MAPE로 영향 축소 + 학교별 p90로 별도 감시.
- **지역 인구 기준선:** f0 드리프트(3.3)·지역 수요 검증에 쓰는 인구는 **주민등록 실적(population_actual)**
  으로 고정. 추계는 대학학령을 유입지 과대·배출지 과소로 왜곡(3.4)하므로 지역 단위 과거 검증 기준으로 부적합.

## 9. 모델 한계 명시 (보정 vs 명시 경계)

**보정(모델이 처리):**
- 대학원 수업료 오염 → ρ_g 환산 분리(1.4).
- 정원외(외국인·편입) → H_ug_out 분리, 정원 델타에서 제외(2.3).
- 휴학·초과·중도탈락 순효과 → m_in·s_k에 흡수(2.1).
- 미충원 학교의 손실 비대칭 → 수요앵커 충원율(3장).
- 입학금 폐지 구조단절 → 5112 기준·5111 외생(6.4).

**명시만(모델이 못 다룸, 주석·비활성):**
- **학년별 정밀 코호트**: KESS에 학년축 없음 → 대학알리미 인증 API 확보 전 불가(2.2d). 현재는 잔존곡선 근사.
- **중도탈락 학교별 실측**: 인증 API 미확보 → 평균값(m_in) 흡수 + 스트레스 슬라이더로만.
- **대학원 정원 정책 연동**: 학부 정원 감축이 대학원 규모에 주는 2차효과 미모형(기본 고정, 선택적 동반감축 슬라이더).
- **국가장학금·등록금 동결 정책 상호작용**: π 시나리오로 외생 반영, 내생 정책반응은 미모형.
- **정원외 외국인 급변**(유학생 유치·급감): 외생 고정, 별도 시나리오로만.
- **인구 자연감소에 의한 f0 하락(정책 무관)**: v1 기본은 f0 고정 → 드리프트 옵션(3.3)으로만. 미충원
  지방대의 "정책과 별개인 충원율 하락"은 기본 화면 밖. 추계 대학학령의 거주지↔등록지 편향은 실적·보정
  (3.4)으로 처리하되 2025+ 미래 구간엔 잔존 불확실.
- **학교 통폐합·캠퍼스 구조조정**: 범위 밖.

**5110 대학원·정원외 혼입 문제 요약 처리방침:** (반드시 UI 주석 노출)

수입"임을 카드에 명시 → ④ ρ_g 민감도(1.2~1.7)를 불확실성 밴드에 포함.

## 10. 구현 스펙 (클라이언트 vs ETL)

### 10.1 역할 분담
- **ETL 사전계산(필수):** 전 학교 패널·외부 데이터·교차연도 디컨볼루션이 필요한 **안정 파라미터**.
- **클라이언트 JS(renderSimulation):** 슬라이더 → 컨볼루션·KPI 재계산·팬차트. 계산량 경미
  (학교당 ≤10년×≤8래그; 전교 집계 344×10×8≈2.7만 연산) → **전부 클라이언트 실시간 가능**, 서버 불요.

### 10.2 dashboard_data.json 추가 필드 — `sim` 블록
