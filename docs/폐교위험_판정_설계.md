# 폐교 위험 판정 레이어 — 설계 (deep-reasoner, 2026-07-20)

시뮬레이션 조건(r·β·π·γ·λ)에 따라 학교별 생존 등급과 "폐교위험 N개교" 집계를 실시간 산출하는 판정 레이어. 모든 임계값은 2016~2024 실제 폐교 대학 패널로 보정했다.

## 0. 실측 근거 (설계 전 확인 완료)

- 적립금총액 latest 344/344 non-null, 적립금지속월수 344/344, corp2024 335/344 → runway 모델 신규 수집 없이 구현 가능. `build/interim/balance_sheet.csv`에 `reserve_total`(9년) + `reserve_principal/discretionary/fund_basic`(2024) 존재 → 가용적립금 haircut 가능.
- 관측창 내 폐교 패널: 4년제 4교(대구외대 2018, 한중대 2018, 한려대 2022, 한국국제대 2023) + 전문대 1(강원관광대 2024) + 대학원대 1(계약신학대학원대 2023). 핵심 검증 N=4~6.
- 폐교 공통 지문: 직전연도 적립금≈0 + 충원율 붕괴 + (대부분) 운영적자. closure_traj 중앙값 충원율 -5년 0.58 → -1년 0.28.
- 기존 위기스코어 결함 실증: 국민대 crisis 55.8("주의") 오분류 — 백분위·정적·시나리오 무반응. → 신모델은 runway 기반.
- closure_traj의 등록금의존율은 폐교교 분모 붕괴로 불안정(4.99, −1.78) → 판정 신호에서 제외.

## 1. 등급 체계

"폐교확실"은 거부 — 모델은 재정 리스크만 측정하며, 폐교는 비리·분규·법인구제 등 재정 외 요인이 개입한다. 4단계 + 미분류:

| 등급 | 조작적 의미 | 색 토큰 |
|---|---|---|
| **안정** (stable) | 흑자 또는 runway ∞ + 충원율 견고 | `--good` |
| **주의** (caution) | 적자여도 완충 장기 / 완충 부재 흑자 / 충원율 약연화 | `--warning` |
| **위험(존속위험)** (atrisk) | 적자로 완충 2~5년 내 소진, 또는 충원율<0.65+적자, 또는 완충부재+적자 | `--serious` |
| **심각(존속위기)** (critical) | 심각적자(<−10%)+완충소진, 또는 충원율<0.5 — 폐교 직전 재무지문 | `--critical` |
| **미분류** (unrated) | 학부미보유(no_ug)·데이터부재·대학원대 | neutral(`--text-muted`) |

헤드라인 = **"폐교위험 N개교" = 위험+심각 합계**. KMU Blue(#004F9F)는 등급 색에 절대 미사용.

## 2. 판정 산식 — 두 축 게이트 매트릭스

자의적 가중합 대신 각 축이 명확한 재무 의미를 갖는 게이트 매트릭스.

- **축1 존속성(flow, 선행):** 정착 운영수지율 M = 시뮬레이션 primed 운영수지 궤적의 마지막 min(3, horizon)년 평균. Healthy(≥+2%) / Thin(0~2%) / Deficit(−10~0%) / Severe(<−10%).
- **축2 완충(stock→runway):** 적립금 소진 rollforward:
```
reserve_{t+1} = reserve_t + primed_운영수지(t) + corp_addl(t)
reserve < 0 되는 첫 해 = 소진예상연도,  R = 소진연도 − t0  (적자 없으면 ∞)
```
Strong(R≥10y) / Moderate(5~10) / Thin(2~5) / Depleted(<2 또는 적립금월수<1).

### 판정 매트릭스

```
             Strong(R≥10)  Moderate(5-10)  Thin(2-5)  Depleted(<2)
Healthy(≥2%)   안정          안정            주의       주의
Thin(0-2%)     안정          주의            주의       위험
Deficit(-10~0) 주의          주의            위험       위험
Severe(<-10%)  주의          위험            위험       심각
```

오버라이드(우선순위 순): ① 충원율<0.50 → 심각 ② 충원율<0.65 → 최소 위험 ③ Severe∧Depleted → 심각(매트릭스 내재) ④ 적립금월수<3 ∧ 흑자 → 안정 금지(주의 상한) ⑤ 적립금월수<1 ∧ 적자 → Depleted 강제.

임계 근거(실측 보정): Healthy 2% = 국민대 2024(2.25%) 안정 유지선. Severe −10% = 한려대(−77%)·강원관광대(−33%)·화성의과학(−34%) 구간. 충원율 0.65·0.50 = closure_traj(-1년 0.28, -3년 0.47)와 대구외대(0.64)·한국국제대(0.81)·한려대(0.38) 분포. runway 2·5 = 상지대 trough R≈0.67(위험) vs 선린대 R≈68(주의) 분리점.

## 3. 입력·전처리

- 운영수지 M: `SIM.recalcSeries(...)`의 primed 운영수지·운영수지율. r=0이면 primed=original=관측 t0값(델타 프레이밍).
- reserve0: 신규 `sim.bySchool[idx].stock.reserve_avail`(ETL A8). 가용 = reserve_total − fund_basic(2024 있으면) 또는 reserve_discretionary; 없으면 reserve_total × 보수 haircut(0.8, A8에서 확정). 결측 시 ext.series.적립금총액 최신 fallback.
- reserve_months: base.stock.reserve_month0 (= 적립금총액/(OP_EX/12)).
- corp_addl(옵션, 기본 0): `max(0, corp2024.revenue − 현행 전입)` min cap — "법인지원 반영" 토글에서만 가산. corp2024.transfer와 5210의 회계 중복도 미검증이므로 보수 산정(가정 — §7).
- fill: f0 = A_in/Q(seg). λ-on이면 f0·(1+λ·cumDrift) 정착값.

## 4. JS 순수 함수 스펙 (sim_model.js 확장, 기존 함수 무수정)

```
SIM.closureGrade(schoolSim, kpiSeries, opts) → {
  grade: 'stable|caution|atrisk|critical|unrated',
  runwayYears: number|Infinity|null, depletionYear: number|null,
  marginEnd, fill, buffer0, reserveMonths,
  drivers: ['deficit'|'no_buffer'|'low_fill'|'severe'|'liquidity'],  // 결정 게이트(설명가능성)
  confidence: 'ok|beta|low', boundaryFlag: bool, gradeRange: [lo,hi]
}
SIM.closureAggregate(allSims, params, opts) → {
  counts: {stable,caution,atrisk,critical,unrated},
  atRiskOrWorse: N, atRiskRange: [lo,hi],
  byRegion: {...}, byScale: {...},
  transitions: { downgraded:N, newAtRisk:N, schools:[{idx,from,to}] }  // 기준(r=0) 대비
}
```

경계밴드: 운영수지'에 ±z·σ(σ=bt.mape_rev, z=1.28) 적용해 재판정 → 등급이 바뀌면 boundaryFlag + gradeRange. runway가 임계 {2,5,10}±1y이면 동일 처리. 344교 × O(1) → 실시간 가능.

## 5. 폐교 사례 역검증 (D6 게이트)

매트릭스를 실측 대입한 사전 결과: 한중대 심각, 대구외대 위험, 한려대 심각, 한국국제대 위험 → **4년제 4/4 민감도 100%**. 강원관광대 심각 포함 5/5. 생존 대조: 총신대(분규·재정건전) 안정, 국민대 안정(기존 오분류 교정), 선린대 주의. 위험 판정된 생존교 상지대(위험)·화성의과학대(심각)는 실제 폐교 직전 수준 재정위기를 법인구제·구조조정으로 모면한 사례 → 오탐이 아니라 모델 범위 밖 요인으로 정직 처리.

검증 절차: 폐교 직전 T-1/T-2 데이터만으로 판정 → 혼동행렬·민감도(게이트 ≥4/5)·특이도(전체 + 분규교 제외 병기)·lead-time·AUC(부트스트랩 CI 병기, N=4~6이라 보조지표). 합격 게이트: 민감도(위험+) ≥0.8 ∧ 명백 건전교 위험+ 0건 ∧ 총신대 ≤주의. leave-one-out 민감도로 임계값 과적합 점검. **D6 합격 전 UI "베타" 라벨.**

## 6. UI 정보위계 (오른쪽 패널)

1. **헤드라인**: "이 조건이면 폐교위험 N개교 (기준 대비 +K)" [심각 X · 위험 Y], 범위 [N_low~N_high].
2. **등급 분포 스택바**(344교): 안정|주의|위험|심각|미분류, 슬라이더 연동.
3. **전이 표시**: 기준(r=0) 대비 하락 K개교.
4. **권역×등급 히트맵**: 지방 소규모 집중 가시화.
5. **학교별 배지 테이블**: 등급칩·소진예상연도·M·fill·β/경계 태그.
6. **검증 참조 패널**: "실제 폐교 5/5 위험+ 판정" + closure_traj 기준선.
7. **정직성 푸터**: 재정위험이지 폐교 확정 아님 · 비리/분규/법인지원 미반영 · 전문대 미검증.

토큰: `--good/--warning/--serious/--critical` + neutral, square-ui pill 재사용. KMU Blue 미사용.

## 7. 리스크·가정

1. **표본 N=4~6** — 통계적 증명이 아닌 보정·정합성 점검으로 프레이밍. AUC 과신 금지.
2. **폐교 ≠ 순수 재정 사건** — 비리(서남대)·분규 폐교, 법인구제 생존(상지대) → 모델은 "재정 폐교위험"의 상·하한.
3. **전문대 시나리오 투영 미검증**(백테스트 코호트 0교) — 관측 스톡 기반 등급은 유효, 투영은 β 배지 분리.
4. **법인여력 데이터 취약**(2024 단일, 폐교교 전부 None) — 기본 runway는 적립금 단독(보수적), 법인지원은 옵션 토글.
5. **reserve_avail haircut**은 2024만 fund_basic 실측 → 타 연도 0.8 계수(가정, A8 확정).
6. 계약신학대학원대는 미분류 가정(no_ug 여부 A9 실측 확인).

## 8. 검토한 대안과 버린 이유

- 가중합 위기지수 확장: 백분위·정적·시나리오 무반응·KMU 오분류 → 게이트 매트릭스로 대체.
- 적립금지속월수 직접 사용: 전체 지출 소각 기준이라 적자 규모 무관 → 적자기반 rollforward 채택(선린대 R≈68 vs 월수 7.3 구분).
- AUC 주지표화: N=4~6에서 CI ±0.2+ → 혼동행렬·민감도·lead-time 주지표.
- 등록금의존율 포함: 폐교교 분모 붕괴로 불안정 → 제외.
- 법인여력 기본 가산: 신뢰 불가 → 옵션 토글.
- "폐교확실" 등급: 모델 보증 불가 → "심각(존속위기)" 명명.
